### Сообщение об ошибке при вводе в поля "Придумайте пароль" и "Повторите пароль", пароля без строчных букв латиницы

## Предусловия

1. {{baseUrl}} взять из файла https://gitlab.com/DJWOMS/junovnettestingdocs/-/blob/main/frontend/base_URL_register.md

## Шаги

1. Зайти на страницу {{baseUrl}} из предусловия

2. Ввести в поле "Придумайте пароль" - "12Q56!AD" (вводить без кавычек "")

3. Ввести в поле "Повторите пароль" - "12Q56!AD" (вводить без кавычек "")

4. Кликнуть на странице за пределами полей для ввода

## Ожидаемый результат

Поле "Придумайте пароль" подсвечивается красным, под полем появляется текст подсказки, красного цвета - "Пароль должен включать минимум одну строчную букву латинского алфавита"
